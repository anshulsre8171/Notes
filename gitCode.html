    -------------------------
            Untracked                     --> new files that git doesn't yet track
            Modified                      --> change file data but not add to git
            Add(Staged)                   --> file is ready to be commited

--------------------------------------------------------------------------------------

    clone & status
    ---------------

        git clone <-some Link->         -->cloning a repository on our local machine.
        git status                      -->display the status of the code.


-------------------------------------------------------------------------------------

    add & commit & push-
    -------------------

    git add <-fileName->  ||  git add .  --> adds new or changed files in your working directory to the git staging area. 
    git commit -m "some message"         --> commit is the record of change
    git push origin main                 --> upload local rep[o content to remote repo.     origin(default) main(branch)
    
-------------------------------------------------------------------------------------------------------------------------
    
    working flow path-
    ------------------
    
    gitHub Repo --> clone --> changes(file me kuch bhi cahnge) --> add --> commit --> push

    Repo --> git clone  --> git status --> git add  --> git commit -m "update msg" --> git push origin main
